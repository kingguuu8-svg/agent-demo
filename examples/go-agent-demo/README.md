# Go Agent Demo

This is a compact Go coding-agent runtime generated by the Rust `agent-demo` from a natural-language specification. It is included as an execution proof: the Rust agent created a second multi-file agent, ran its toolchain, and iterated on failures without an agent framework.

## Run

Go 1.21+ and a real DeepSeek key are required. The key is read only from the environment.

```text
set DEEPSEEK_API_KEY=your-key
go run .
```

Useful commands are `tools`, `memory`, `history`, `clear`, and `exit`. Validate the project with:

```text
gofmt -w .
go vet ./...
go test ./...
go test -run TestLiveDeepSeekAgentLoop -v
```

## Runtime Design

`LLMAgent.Run` builds an OpenAI-compatible request containing the system prompt, registered JSON tool schemas, and the current goal. DeepSeek either returns text or tool calls. The runtime executes each call, appends tool observations to the active message chain, and loops until a final answer or the 20-step guard.

Eight tools demonstrate registration: `calculator`, `read_file`, `write_file`, `edit_file`, `list_dir`, `shell`, mock `search`, and `finish`. They run with the current process's host permissions; this proof project deliberately has no sandbox or approval UI.

## Memory Placement

`Memory` is a mutex-protected bounded ring of 200 goal, LLM, and tool observations. Goals and model text are recorded after they occur; tool results are recorded immediately after execution. `memory` displays the store and `clear` removes it. Within one `Run`, the authoritative context is `LLMAgent.messages`: every tool observation is placed directly after its matching assistant tool call before the next LLM request.

This generated proof resets `LLMAgent.messages` for each new goal and does not feed the observation log back into later goals. It therefore demonstrates active-loop context placement, not durable recall. Persistent isolated sessions, resume history, SQLite recall, and context compression are implemented and documented by the requirements-complete Rust runtime at the repository root.
